Project: "Steakz" Full-Stack Management System
Role: Senior Full-Stack Developer & UI Designer.
Objective: Build a web application named "Steakz" that replicates the specific UI/UX, color palette, and data structure shown in the provided screenshots.

1. Visual Style & Color Architecture
Strictly implement the following color tokens and design rules:

Landing/Login Background: Use a high-quality, blurred image of a cooked steak.

Primary Brand Green: #2d5a43 (Used for full-page backgrounds and headers).

The "Olive" Module: #8b864e (Used for the Login card and Dashboard buttons).

UI: A centered #8b864e box over a steak image background.

Inputs: Stark white boxes for "Email address" and "Password."

Elements: Include a "Check me out" checkbox and a green "Log In" button.

Logic: Build a form on a #2d5a43 background.

Fields: Customer Name, Phone Number.

Menu Items: * Steak: (Strip, T-bone, Filet, RibEye)

Sub: (Fries, Salad, Grilled Potatoes)

Drinks: (Soda, Water)

Action: A "Confirm Order" button that saves data to a local state or database.

Module C: Admin & Order Management
Table View: A "Previous Orders" list with columns: Order Number (001, 002...), Name, Phone, Steak, Qty, Sub, Qty, Drinks, Qty, Total.

Navigation: Use the colored square-button layout (Add Employee, Delete, Previous Orders, Dashboard) as seen in the references.
